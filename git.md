# Git

1. git 시작하기

   ```sh
   $ git init
   ```

   이제부터 해당 디렉토리를 git으로 관리하겠다!

2. 상태 확인하기

   ``` sh
   $ git status
   ```

3. 스테이지에 올리기

   ```sh
   $ git add 파일명
   ```

4. 커밋하기(사진찍기)

   ```sh
   $ git commit -m '메세지'
   ```

5. 기록 확인하기

   ```sh
   $ git log
   ```

6. 리모트(원격 저장소) 등록하기

   ```sh
   $ git remote add origin 원격 저장소 주소
   ```

   shift + insert : 복사

7. 파일 푸시하기(업로드하기)

   ```sh
   $ git push origin master
   ```
